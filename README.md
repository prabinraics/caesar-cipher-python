# Caesar Cipher (Python)

## Description
This project is a Python-based implementation of the Caesar Cipher algorithm, a basic encryption technique used in cryptography. It allows users to encrypt and decrypt messages using a shift value.

## Features
- Encrypt plain text messages
- Decrypt encrypted messages
- User input-based system
- Simple command-line interface

## Technologies Used
- Python

## How It Works
The Caesar Cipher shifts each letter in the message by a fixed number of positions in the alphabet.

Example:
Input: HELLO (shift = 3)  
Output: KHOOR

## How to Run
1. Open the project folder
2. Run the Python file:
   python main.py
3. Enter your message
4. Choose encrypt or decrypt
5. Enter shift value
